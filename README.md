mesos集群监控脚本
================================

系统需求
--------------------------------
操作系统：Linux

主要逻辑
--------------------------------
获取mesos集群API接口数据，解析返回结果，将key组装成json后push到falcon-agent
接口解释请参照:
     http://mesos.mydoc.io/

使用方法
--------------------------------
1. 根据实际部署情况，配置采集master或slave数据;
2. 测试： ./control build && ./control start
